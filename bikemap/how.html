Easy-Travel Backend

This page uses a google maps api, which is I don't want to upload to GitHub, so its hidden in a file named "config.js" as "window.config={keyGoesHere}" that is hidden with a .gitignore file.

In order to use the map fuctionality you need make a local copy of the repository and place the "config.js" and .gitignore files in the bikemap folder on the root of the folder.

Side note this doesn't protect the API key from people scanning the network, and this sight is just a proof of concept, so becareful where you run it
